# Lifestyle-and-Sleep-Heath-Clusters-Analysis
Uma aplicação de aprendizado de máquina, no contexto de “Qualidade de Sono e Estilo de Vida”, analisando dados  e relacionando-os com ferramentas de análise. Será utilizado um método de aprendizado não supervisionado, com clusterização, através do algoritmo K-médias.  A base de dados utilizada é do site Kaggle. Mais informações no README.
